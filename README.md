# Viajes Increíbles - Aplicación Web de Recomendación de Viajes

Este es un sitio web estático diseñado para ofrecer recomendaciones de viaje. Incluye información sobre destinos fascinantes, una descripción de la empresa y un formulario de contacto.



## Criterios del proyecto

El proyecto cumple con los siguientes puntos clave:

1.  **Página de inicio con introducción**: Contiene una página (`index.html`) con una clara introducción que motiva al usuario a explorar el sitio.
2.  **Barra de navegación funcional**: Incluye una barra de navegación en todas las páginas con enlaces a Inicio, Nosotros y Contacto. Es completamente responsiva para dispositivos móviles.
3.  **Página "Nosotros"**: Existe una página (`nosotros.html`) que describe la misión y visión de la empresa.
4.  **Página de "Contacto" con formulario de correo electrónico**: Contiene un formulario funcional (`contacto.html`) para que los usuarios puedan enviar mensajes.
5.  **Recomendación de Playa, Templo y País:
    - **Playa**: Se recomienda **Playa del Carmen, México** con dos espacios designados para imágenes.
    - **Templo**: Se recomienda el **Templo de Angkor Wat, Camboya** con dos espacios designados para imágenes.
    - **País**: Se recomienda **Santorini, Grecia** con dos espacios designados para imágenes.

## Estructura del Proyecto

El proyecto se organiza en la siguiente estructura de archivos:

```
Aplicacion Web de Recomendacion de Viajes/
├── index.html              # Página de inicio con recomendaciones
├── nosotros.html           # Página "Nosotros" con Misión y Visión
├── contacto.html           # Página de "Contacto" con formulario
├── css/
│   └── style.css           # Estilos personalizados para el sitio
├── js/
│   └── app.js              # Lógica de la aplicación (menú móvil y formularios)
├── imagenes/               # Carpeta vacía destinada a las imágenes de destinos
└── README.md               # Este archivo
```

## Tecnologías Utilizadas

- **HTML5**: Para la estructura semántica de las páginas.
- **CSS3**: Para el diseño visual, los estilos responsivos y las animaciones.
- **JavaScript**: Para la funcionalidad interactiva, como el menú de navegación móvil y el manejo del formulario de contacto.

## Instalación y Uso

Para ejecutar este proyecto localmente:

1.  Clona o descarga el repositorio.
2.  Navega hasta la carpeta del proyecto.
3.  Abre el archivo `index.html` en tu navegador web favorito.

## Créditos

Desarrollado en Fedora 44 como parte de un proyecto de desarrollo web frontend.
